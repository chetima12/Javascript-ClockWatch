# Javascript-ClockWatch

Une petite application web qui affiche une horloge créée avec HTML, CSS et JavaScript.

## Aperçu
Javascript-ClockWatch est un projet simple pour afficher l'heure locale dans le navigateur. Il contient une page HTML, un script JavaScript pour gérer l'heure et du style CSS pour l'apparence.

## Fonctionnalités
- Affichage de l'heure en temps réel
- Mise à jour automatique toutes les secondes
- Styles CSS pour une présentation claire et responsive
- Code simple et facile à personnaliser

## Structure du dépôt
- `index.html` — page web principale à ouvrir dans le navigateur.
- `script.js` — logique JavaScript qui met à jour l'heure et gère l'affichage.
- `style.css` — styles pour l'horloge et la mise en page.

## Installation et usage
1. Cloner le dépôt ou télécharger les fichiers :
   - git clone https://github.com/chetima12/Javascript-ClockWatch.git
2. Ouvrir `index.html` dans votre navigateur (double‑clic ou via un serveur local).
3. L'horloge démarre automatiquement et se met à jour chaque seconde.

Optionnel : pour un développement local avec rechargement automatique, servez le dossier avec un petit serveur (par exemple `npx http-server` ou l'extension Live Server de VS Code).

## Personnalisation
- Modifier `style.css` pour changer les couleurs, tailles et la disposition.
- Modifier `script.js` pour changer le format de l'heure (12h/24h), ajouter des fuseaux horaires ou des fonctions supplémentaires (alarme, chronomètre).
